Sobre Vendas em JSF Eclipse jBoss Este proyecto tiene nada mas la finalidad de contribuir con algunos código orientado a objetos utilizando las tecnologias mas importantes que abarca o que soporta el JSF utilizando el ambiente de programación Eclipse Luna con las tecnoligias de soporte tales como jBoss, Ajax, Hibernate, Jasper y utilizando el repositorio E-Git que facilita la utilización de repositorio on-line como Git-Hub.
Queda claro que el projecto es nadas mas con el fin de colaborar con algunos colegas nuevos que quieren adentrarse en la programción orientada a objetos JAVA, en este caso JSF en aplicación WEB.
Tambien si consiguen analizar y entender el projecto creo que serviría como estructura base para cualquier projecto que quieran iniciar, ja que esta testado y comprobado que raliza perfectamente los CRUD para los tres entidades que utiluza(Cliente,Produto y Venda), tambien dejo en el un ejemplo de utilización de iRepor generado en el propio navegador en pdf listo para que el usuario pueda imprimir directamente o descargar y guardar.
